## 📖 Overview

CompileFlow is a web-based compiler simulator that visualizes the complete compilation process. It enables students and developers to understand how source code is transformed into executable code through each phase of compiler design.

The project provides an intuitive interface that demonstrates lexical analysis, syntax analysis, semantic analysis, intermediate representation (IR) generation, optimization, and final assembly code generation.

---

## ✨ Features

- 🔹 Lexical Analysis (Tokenizer)
- 🔹 Syntax Analysis (Parser)
- 🔹 Semantic Analysis
- 🔹 Parse Tree Generation
- 🔹 Intermediate Representation (IR) Code
- 🔹 Code Optimization
- 🔹 Assembly Code Generation
- 🔹 Compilation Statistics
- 🔹 Interactive UI
- 🔹 Real-Time Processing
